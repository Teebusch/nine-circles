
# Nine Circles

A classic capture the hill card game.
## How to play?

Place 3 troops under each circle that trump your oppoonents troops.
The first player to capture 5 circles or 3 circles in a row wins.

### Formations

* Consecutive numbers, same color
* Same number
* Same color
* Consecutive numbers
* Sum of cards
